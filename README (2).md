## 📌 Project Overview

This project demonstrates how prompt engineering can be used to generate a complete UGC (User Generated Content) style ad pack for a D2C skincare product. Instead of polished, salesy ad copy, the goal was to produce content that sounds like a real customer talking about a product they genuinely like — which is what actually performs well on Instagram Reels, Shorts, and Facebook ads today.

---

## 💡 Why This Product?

I picked an acne/pimple face wash because skincare is one of the biggest UGC categories — almost every D2C skincare brand runs creator-style ads instead of traditional polished commercials, because people trust "real" testimonials over obvious advertising. Acne specifically has a strong emotional angle (frustration, self-consciousness, relief after finding something that works), which makes it a good test case for writing authentic, non-salesy hooks.

## 🧠 Prompt Engineering Strategy

| Step | Purpose |
|---|---|
| Hook Generation | Create multiple scroll-stopping opening lines for the same product |
| Script Writing | Generate full UGC-style scripts following problem → solution → CTA |
| Platform Adaptation | Adjust tone/length for Instagram Reels, Facebook, and YouTube Shorts |
| Caption Writing | Generate short captions to pair with each script |

## ✅ What Was Built

1. 5 scroll-stopping hooks for the same product
2. 3 full UGC ad scripts (problem → solution → CTA format)
3. Platform-adapted versions (Instagram Reels, Facebook, YouTube Shorts)
4. Matching captions for each script
5. CTA variations for different funnel stages

---

## My Observations

- My first attempt at hooks sounded too much like ad copy — phrases like "transform your skin" gave it away immediately. I had to explicitly tell the AI to avoid marketing language and write like a frustrated customer instead
- The problem → solution → CTA framework worked well, but I had to ask for a believable "before" state (acne frustration) before jumping to the product, otherwise it felt rushed
- Platform adaptation mattered more than I expected — the same script needed to be noticeably shorter and punchier for YouTube Shorts vs. a slightly longer Instagram Reel voiceover
- This prompt system is reusable for almost any D2C product — skincare, supplements, gadgets — by just swapping the product details and pain point

---

## ⚠️ Limitations

- This is fictional ad content for a fictional brand (ClearSkin) created for internship purposes — not a real client
- Any real skincare ad claims would need legal/compliance review before running as paid ads (skincare advertising has restrictions in most countries)
- Hooks and scripts are AI-generated starting points — a real UGC creator would still need to film and deliver them naturally

---

## 🔧 How to Reuse This Prompt System

1. Replace the **product/brand name**
2. Replace the **core pain point** (acne → dry skin, dullness, hair fall, etc.)
3. Keep the same hook → script → platform adaptation → caption structure
4. Run the same prompts — outputs adapt automatically
